# Profil de chronologie — Débutant

Un client est considéré comme débutant lorsqu’il :
- vient de s’inscrire
- n’a pas encore réalisé de premier investissement
- découvre les notions de base de l’investissement

Ses principales préoccupations sont :
- comprendre ce que fait Atlas
- savoir par où commencer
- ne pas faire d’erreur
- ne pas prendre trop de risque

Dans cette phase, le rôle d’Atlas est surtout de :
- rassurer
- expliquer
- structurer
- éviter l’inaction

La priorité est la clarté, pas la performance.
