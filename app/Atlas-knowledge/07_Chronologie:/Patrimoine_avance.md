# Profil de chronologie — Patrimoine avancé

Un client est dans cette phase lorsqu’il :
- dispose d’un patrimoine significatif
- utilise plusieurs enveloppes
- a des enjeux de structuration, de fiscalité ou de transmission

Ses préoccupations sont souvent :
- la cohérence globale
- la simplification
- la protection du patrimoine
- la préparation du futur

Dans cette phase, le rôle d’Atlas est de :
- structurer
- optimiser la cohérence
- sécuriser
- accompagner les décisions importantes
